# CUSTOMER-BUYING-BEHAVIOUR-ANALYSIS

Analyzed customer purchasing patterns using Microsoft Excel by applying data cleaning, Pivot Tables, charts, and dashboards to identify sales trends, customer preferences, and buying behaviour insights.

## Objective

The Customer Buying Behaviour Analysis project aimed to analyze customer purchasing patterns using Microsoft Excel to identify trends, preferences, and spending behaviour across different customer segments.

The project focused on transforming raw sales data into meaningful insights through data cleaning, Pivot Tables, charts, and interactive dashboards to support better business decision-making.

---

### Skills Learned

- Data cleaning and preprocessing in Excel.
- Using Pivot Tables and Pivot Charts for analysis.
- Customer trend and sales pattern analysis.
- Dashboard creation and data visualization.
- Business insight generation from sales data.

---

### Tools Used

- **Microsoft Excel**
- **Pivot Tables**
- **Pivot Charts**
- **Excel Functions & Formulas**

---

## Key Insights

- Identified top-performing products and categories.
- Analyzed customer spending behaviour across demographics.
- Visualized sales trends and purchasing patterns.
- Created interactive dashboards for business reporting.

---

## Outcome

Successfully developed an Excel-based interactive dashboard that provided actionable insights into customer purchasing behaviour and sales performance.
